# CRUD moderno y responsive

Esta carpeta contiene un CRUD hecho con **HTML + CSS + JavaScript vanilla**.

## Funcionalidades
- Crear tareas.
- Leer/listar tareas.
- Editar tareas.
- Eliminar tareas.
- Búsqueda por título.
- Persistencia local con `localStorage`.
- Diseño responsive.

## Uso
1. Abre `index.html` en tu navegador.
2. Agrega, edita o elimina tareas.
3. Los datos quedan guardados en tu navegador.
